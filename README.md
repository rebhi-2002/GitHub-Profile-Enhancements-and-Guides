## Markdown Basics
- [CommonMark](https://commonmark.org/help/): A reference for Markdown basics.
- [GitHub's Guide to Markdown](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax): GitHub's guide to using Markdown.
- [Markdown-it](https://markdown-it.github.io/): A powerful Markdown tool.
- [Daring Fireball Markdown Syntax](https://daringfireball.net/projects/markdown/syntax): The original Markdown syntax guide.
- [CommonMark Dingus](https://spec.commonmark.org/dingus): A tool to test Markdown files.
- [Markdown Tutorial](https://www.markdowntutorial.com/): An interactive Markdown tutorial.

## Markdown Editors
- [Marxi Demo](https://marxi.co/#fnref:demo): Direct Markdown editing with instant preview.
- [Markdown Live Preview](https://markdownlivepreview.com/): Real-time preview while editing Markdown.
- [Editor.md](https://pandao.github.io/editor.md/en.html): A powerful Markdown editor.
- [Dillinger](https://dillinger.io/): An online Markdown editor.

## GitHub-Specific Markdown
- [GitHub Flavored Markdown](https://github.github.com/gfm/): GitHub's Markdown extensions.
- [GitHub's Advanced Formatting](https://docs.github.com/en/get-started/writing-on-github/working-with-advanced-formatting): Advanced formatting on GitHub.
- [GitHub Markdown Cheatsheet](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet): A handy cheat sheet for Markdown on GitHub.

## Markdown in Visual Studio Code
- [VS Code Markdown Support](https://code.visualstudio.com/docs/languages/markdown/): Markdown support in Visual Studio Code.

## Readme Enhancement

## Icons and Logos

- [Vector Logo Zone](https://www.vectorlogo.zone/logos/): A library of project logos.

## Shields and Badges
- [Shields.io](https://shields.io/): Create badges and shields for your projects.

## GitHub Profile Enhancement

### Profile Overview
- [GitHub Profile Trophy Generator](https://rahuldkjain.github.io/gh-profile-readme-generator/): Create awards for your GitHub profile.
- [GitHub Profile Achievements](https://www.libhunt.com/compare-github-profile-trophy-vs-github-profile-achievements): Showcase achievements on your GitHub profile.
- [GitHub Profile Maker](https://github-profile-maker.vercel.app/): Customize the appearance of your GitHub profile.
- [GitHub Readme Stats](https://github.com/anuraghazra/github-readme-stats): GitHub profile statistics.
- [Profile Summary for GitHub](https://github.com/tipsy/profile-summary-for-github): Comprehensive summary for your GitHub profile.

### Achievements and Awards
- [GitHub Profile Achievements](https://github.com/Schweinepriester/github-profile-achievements): Display your achievements.
- [GitHub Profile Trophy](https://github.com/ryo-ma/github-profile-trophy): Showcase trophies on your profile.

### GitHub Stats
- [Get GitHub Achievements](https://github.com/4xmen/Get-Github-Achievements): Collect achievements from your GitHub profile.
- [GitHub Profile 3D Contributions](https://github.com/yoshi389111/github-profile-3d-contrib): Display contributions in 3D.
- [GitHub Profile Summary Cards](https://github.com/vn7n24fzkq/github-profile-summary-cards): Display summary cards for your GitHub profile.
- [Waka Readme Stats](https://github.com/anmol098/waka-readme-stats): Time spent statistics on GitHub.
- [GitHub Profile Views Counter](https://github.com/antonkomarev/github-profile-views-counter): Count views on your GitHub profile.
- [GitHub Readme Streak Stats](https://github.com/DenverCoder1/github-readme-streak-stats): Display activity streaks on your profile.

## Markdown Guides

## GitHub Writing and Formatting Guides
- [GitHub's Guide to Writing and Formatting](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/about-writing-and-formatting-on-github): GitHub's guide to writing and formatting content.
- [GitHub's Quickstart for Writing on GitHub](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/quickstart-for-writing-on-github): A quick guide to writing on GitHub.
- [How to Format Code in Markdown](https://www.freecodecamp.org/news/how-to-format-code-in-markdown/): How to format code using Markdown.
- [Update GitHub Profile Trophy](https://dev.to/ryoma/update-github-profile-trophy-2751): How to update your GitHub profile trophies.
- [Excellent GitHub Profile: Stat Streak and Trophies](https://learn.adafruit.com/excellent-github-profile/stat-streak-and-trophies): How to enhance your GitHub profile.
- [Add Dynamically Generated GitHub Trophy](https://morioh.com/a/d05bc0c0ddd4/add-dynamically-generated-github-trophy-on-your-readme): Add dynamically generated GitHub trophies to your README.
